CSS Frameworks / Pattern Libraries

Bootstrap

Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Categories: frameworks. Pricing: Free

Material Design Lite

Material Design Lite lets you add a Material Design look and feel to your websites. It doesn’t rely on any JavaScript frameworks and aims to optimize for cross-device use, gracefully degrade in older browsers, and offer an experience that is immediately accessible.

Categories: frameworks. Pricing: Free

Bourbon

A lightweight semantic grid framework for Sass with Bourbon.

Categories: frameworks, SASS. Pricing: Free

Materialize

A modern responsive front-end framework based on Material Design.

Categories: frameworks. Pricing: Free

Material Kit

Material Kit is a Free Bootstrap UI Kit with a fresh, new design inspired by Google's material design.

Categories: frameworks. Pricing: $79

Kube

Minimalism and efficiency. Kube is simple yet beautiful, helping you focus on your goals, not on your tools.

Categories: frameworks. Pricing: Free

Wing

Wing is made for those smaller side projects, as it isn't as full featured as Bootstrap, Foundation, etc. Wing styles basic elements, comes with a grid, and some fading animations. Still, what more do you need?

Categories: frameworks. Pricing: Free

Hack CSS

The CSS framework for hackers. Inspired by markdown.css and zeit.co

Categories: frameworks. Pricing: Free

Mailchimp's Pattern Library

The MailChimp Pattern Library reflects their move to a more responsive, nimble, and intuitive app. Mailchimp has worked to distill most of the MailChimp interface into a set of atomic pieces to speed up workflows and solve some internal communication problems.

Categories: pattern libraries. Pricing: Free

Lightning Design System

Lightning Design System is Salesforce's platform-agnostic CSS framework/pattern library.

Categories: pattern libraries. Pricing: Free
